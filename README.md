# Image Captioning using BLIP

This project generates captions for images using the Salesforce BLIP model and Gradio interface.

## Live Demo

Try the application live:
[![Live Demo](https://img.shields.io/badge/Live-Demo-green)](https://huggingface.co/spaces/Purnapragya/image_captioning_app)

## Installation

pip install -r requirements.txt

## Run

python app.py

## Model Used

Salesforce/blip-image-captioning-base

## Features

- Image upload
- AI caption generation
- Gradio web interface
